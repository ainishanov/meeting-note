How to Install a Meeting Transcription App on Windows

Set up Meeting Note, connect your API keys, and test recording before your next call.

Download First Open Source Code

Requirements

  • Windows 10 or Windows 11.
  • An audio output device that supports WASAPI loopback.
  • An OpenAI API key for speech-to-text.
  • An OpenRouter-compatible API key for summaries.

Install steps

  1. Download the latest Windows installer.
  2. Run the setup wizard and choose whether to add a desktop shortcut.
  3. Open Meeting Note from the Start menu.
  4. Open settings and add your API keys.
  5. Record a short test call and confirm the transcript appears in history.

First test recording

Use a short non-sensitive audio test before a real meeting. Confirm that system audio is captured, microphone mixing behaves as expected, and the transcript search finds the test phrase.
