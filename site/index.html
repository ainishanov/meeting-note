Local-first workflow

Your meetings stay on your computer.

Meeting Note stores recordings, transcripts, summaries, logs, and its SQLite database locally under the app data folder.

You control the API keys. The app uses your OpenAI key for transcription and your OpenRouter-compatible key for summaries.

Questions

Know this before you install.

Is Meeting Note free?

Yes. The app is free and open source. API usage is billed by the providers you connect.

Which operating system is supported?

Meeting Note is built for Windows 10 and Windows 11.

Can I suggest improvements?

Yes. Use GitHub Issues to request features, report hardware problems, or discuss workflow gaps.
