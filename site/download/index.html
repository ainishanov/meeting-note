What happens after download

  1. Run the downloaded installer.
  2. Keep the default per-user installation folder or choose another one.
  3. Add your OpenAI key for transcription and your OpenRouter-compatible key for summaries.
  4. Start a short test recording before using it in a real meeting.

If Windows shows SmartScreen

Meeting Note is early open-source software, so Windows may warn that the app is not commonly downloaded yet. Check the release page, verify the SHA256 hash, and run it only if you trust the source.
